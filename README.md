warp-dl is a simple downloader application written in Go.

## Features

- Download files from the internet via HTTP/HTTPS
- Command-line interface
- Fast and efficient

## Requirements

- Go 1.18 or newer

## Installation

Clone this repository and build the binary:

```sh
git clone https://github.com/yourusername/warp-dl.git
cd warp-dl
go build -o warp-dl
```

## Usage

```sh
./warp-dl <url> [destination]
```

Example:

```sh
./warp-dl https://example.com/file.zip ./file.zip
```
